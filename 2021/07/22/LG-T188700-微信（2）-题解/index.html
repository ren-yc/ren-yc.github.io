洛谷题目地址

题目解析

这是一道典型的 C++ 结构体排序题。

要点：题目要求从大到小排序，$a$ 一样的按 $name_0$（字符串首） 的 ASCII 码从小到大排序，所以我们还要自己写 cmp 函数，来比较两结构体。

代码分析

输入正整数 $n$，然后循环输入 $a_i$。

建议使用 sort 函数偷懒，写一个 cmp。

出题人 NB！！！不卡我们 long long 了

代码

#include <bits/stdc++.h>
using namespace std;

struct node {
    string name;
    int a;
} wx[101];

bool cmp(node a, node b) {
    if (a.a == b.a) {
        return a.name[0] < b.name[0];
    }
    return a.a > b.a;
}

int main() {
    int n;
    cin >> n;
    for (int i = 0; i < n; i++) {
        cin >> wx[i].name >> wx[i].a;
    }
    sort(wx, wx + n, cmp);
    for (int i = 0; i < n; i++) {
        cout << wx[i].name << endl;
    }
    return 0;
}
