洛谷

UVa Online Judge

题目大意

一只机器人站在一条线的零点，LEFT 命令使它向左走一格，RIGHT 命令则是向右走一格，求机器人最终所在的位置。

解题方法

我们可以定义一个 $a$ 数组，用于存放命令，且只能有 $-1$ 和 $1$ 两个值，分别代表往左和往右。

读入的话可以写一个函数，使用 Python 数组的 append 函数在数组末尾追加数值。

代码

详细解释在注释中。

a = []

def f(c):#输入函数
    if c == "LEFT":#判断
        a.append(-1)#追加
        return -1
    elif c == "RIGHT":
        a.append(1)#追加
        return 1
    else:
        a.append(a[int(c) - 1])#追加
        return a[-1]

t = int(input())
for _ in range(t):#循环
    m = int(input())#输入
    a = []#清空数组
    now = 0#清零
    for i in range(m):
        k = list(input().split())
        if len(k) == 1:
            now += f(k[0])#移动
        else:
            now += f(k[2])#同上
    print(now)#输出

$$END$$
